### 1102 개인과제 (제출기한 : 11/5<일> 오후 12시)

아래 두가지 프로그램 중 하나를 작성해 결과화면과
함께 소스코드제출

1. 숫자야구게임
컴퓨터가 3자리 수를 랜덤하게 뽑아낸다. (각 숫자는 서로 다르다)
사용자에게 1~9 중 3자리 수를 입력받는다. (각 숫자는 서로 다르다)
만약 컴퓨터가 236이라는 숫자를 뽑아냈다면...
사용자가 입력한 수가 265라면 1스트라이크 1볼
사용자가 입력한 수가 623라면 3볼
입력을 10회 받아서 10회 안에 3스트라이크를 만든다면
"9회만에 야구게임에서 승리하셨네요!"를 출력한다.
만약 10회를 초과하여 3스트라이크를 만든다면
"야구게임 패배! 다시하시겠습니까?(Y/N)"을 출력하고 Y또는 N을
입력받는다.
사용자가 Y를 입력하면 야구게임이 다시 시작되고 N을 입력하면
야구게임이 종료된다. System.exit(0);

2. 로또645게임
1~45까지의 중복되지 않는 임의의 번호 7개를 뽑는다.
(7번째 나온 번호는 보너스번호)
<등수>
1등 : 최초 6개번호랑 사용자가 입력한 6개번호가 모두맞음
2등 : 최초 5개번호랑 보너스번호가 사용자가 입력한 6개번호가 모두맞음
3등 : 최초 5개번호가 사용자가 입력한 6개번호에 있음
4등 : 최초 4개번호가 사용자가 입력한 6개번호에 있음
5등 : 최초 3개번호가 사용자가 입력한 6개번호에 있음
나머지 꽝~!
